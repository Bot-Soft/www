BotSoft Script

The BotSoft Script is a JSON file for bots definition. Each element in the file represents a block. The key of the block is the block name. The value is the block's steps. The steps support all of the available messages that Facebook Messenger Platform provides. {% highlight JavaScript %} ... "welcome": { "steps": ["text": "hello, world!"] } ... {% endhighlight %}
